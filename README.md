# Проектирование информационной архитектуры и карт экранов (UX/UI)

Репозиторий содержит результаты этапа системного и UX-анализа: карты экранов (Screen Maps) для мобильной и десктопной версий веб-приложения. Разработка велась на стыке бизнес-требований и технической реализации (API).

## Входные данные и методология
Проектирование велось итеративно на основе следующих этапов:
1. **Сбор требований:** Проведение серии интервью с заказчиком, выявление бизнес-логики и ограничений системы.
2. **Декомпозиция (USM):** Разработка User Story Map в Miro для фиксации пользовательских путей (User Flows) и MVP-функционала.
3. **Архитектура интерфейса:** Формирование структуры экранов с учетом будущей архитектуры клиент-серверного взаимодействия (коллекций API-запросов).

## Спецификация проекта
* **Артефакт:** Интерактивная карта экранов (Screen Map)
* **Платформы:** Mobile UI, Desktop Web UI
* **Инструментарий:** Figma, Miro
